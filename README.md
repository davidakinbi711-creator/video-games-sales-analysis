# Video Games Sales Analysis 1977–2020

## Overview
Analysis of 18,832 video games across 40 years of gaming history 
to understand what drives commercial success in the video games industry.

## Business Questions
1. Does critic score predict sales?
2. Which genres dominated each decade?
3. Who are the most dominant publishers?
4. Where do critic scores and commercial success diverge?

## Key Findings
- Higher critic scores positively correlate with higher sales
- Action and Shooter genres have dominated since the 2000s
- Activision leads all publishers by total sales volume
- Significant outliers exist where franchise loyalty outweighs critic scores

## Tools Used
- MySQL — data querying and analysis
- Tableau Public — data visualisation and dashboard
- Excel — data storage and preparation

## Files in This Repo
- `queries.sql` — all SQL queries used in the analysis
- `score_vs_sales.csv` — critic score vs sales results
- `genre_decades.csv` — genre performance by decade
- `publishers.csv` — top 20 publishers by sales
- `gems_overrated.csv` — hidden gems and overrated games analysis
